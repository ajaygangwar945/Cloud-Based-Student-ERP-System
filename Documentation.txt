Step 14: Create Jenkins Pipeline

Create:

Jenkinsfile

Pipeline stages:

1. Pull Code
2. Install Dependencies
3. Run Tests
4. Build Docker Image
5. Push Docker Image
6. Deploy to Kubernetes

Example pipeline:

pipeline {
    agent any

    stages {
        stage('Build') {
            steps {
                sh 'npm install'
            }
        }

        stage('Docker Build') {
            steps {
                sh 'docker build -t student-erp .'
            }
        }

        stage('Deploy') {
            steps {
                sh 'kubectl apply -f k8s/'
            }
        }
    }
}
